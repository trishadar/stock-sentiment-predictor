# Stock Sentiment Predictor

**Description:**  
A web app that analyzes the latest news articles for any stock ticker and provides a sentiment-based trading suggestion (Buy, Hold, or Sell). The backend uses **VADER sentiment analysis** on news content, and the frontend displays headlines with their sentiment scores.

**Live Demo:**  
[View Live Demo](https://trishadar.github.io/stock-sentiment-predictor/)

**Tech Stack:**  
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python, Flask  
- **APIs:** NewsAPI (for fetching stock news)  
- **Libraries:** VADER Sentiment, Requests, Flask-CORS
